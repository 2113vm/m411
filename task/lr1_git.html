Лабораторная работа №1 Git. Установка и настройка.

Цель работы:

  • Установить git и подготовить его к работе.

Установите git.

  • С сайта https://git-scm.com/
  • Откройте консоль git и настройте отображаемое имя пользователя и пароль
    • git config --global user.name "Your Name"
    git config --global user.email "your_email@whatever.com"
  • git status git add task git add --all git commit -m "Initial commit" git commit -m "Add name at 10" gitk --all
